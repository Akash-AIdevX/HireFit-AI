# HireFit AI

HireFit AI is an AI-powered SaaS platform that helps job seekers improve their resumes using ATS analysis. Users can upload a resume, compare it against a job description, receive AI-generated feedback, and manage subscriptions through Stripe.

## Features

- User Authentication (JWT)
- Resume Upload (PDF)
- AI-Powered ATS Analysis (Google Gemini)
- Resume Review History
- Dashboard & Analytics
- Stripe Subscription & Billing
- Secure PostgreSQL Database (Neon)
- Fully Deployed (Vercel + Railway)

## Tech Stack

**Frontend**
- Next.js
- TypeScript
- Tailwind CSS

**Backend**
- FastAPI
- SQLAlchemy
- PostgreSQL (Neon)
- JWT Authentication
- Stripe API
- Google Gemini API

## Deployment

- Frontend: Vercel
- Backend: Railway
- Database: Neon PostgreSQL
